EVENT_HANDLERS

this module adds simpler event handling to your ruby code each slot should have for parameters: (event) add a slot with this:

  • addHandler(object,:signalName,:slotName)

and also include this module into each class in which you want to use this :-) this is not needed if you’re using an AGWidget or AGApplication object
