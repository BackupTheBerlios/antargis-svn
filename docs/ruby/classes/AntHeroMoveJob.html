Class AntHeroMoveJob
In: ruby/ant_hljobs.rb
Parent: AntHLJob

This is the High-level move job for heroes. it uses the path-finding-algorithm provided by the map-object. it runs in several stages: 1) formatting the troops 2) go to next waypoint - and go back to 1), if necessary 3) moveFinished is set to true and eventMoveFinished is called
