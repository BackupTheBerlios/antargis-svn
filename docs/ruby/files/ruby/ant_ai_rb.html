BoA AI

The AI within BoA is based upon the AntBasicAI class. So every AI you write has to be derived from this. Within such an implementation you’re allowed to use an interface, which is defined in AIInterface. For more information on this refer to files/ruby/ant_ai_interface_rb.html.

For more detailed information on how to program an AI go to files/ruby/ant_ai_interface_rb.html or to ../tutorials/ai_scripting.html
